# Stardew Valley Farm Planner

A free online Stardew Valley farm layout design tool that helps players plan and manage their farms.

## About This Project

Stardew Valley Farm Planner is a free online farm layout design tool that supports multiple farm maps, indoor decoration, furniture placement, seasonal switching, save import/export, and more. It allows players to easily plan their ideal farm layouts.

## Key Features

- **Multiple Farm Maps** - Supports Standard Farm, Forest Farm, Wilderness Farm, Hill-top Farm, Beach Farm, Meadowlands Farm, Riverland Farm, Ginger Island West, Four Corners Farm, Big House, and more
- **Indoor Decoration** - Complete indoor design and furniture placement system
- **Furniture System** - Supports various furniture and decoration items placement and management
- **Seasonal Switching** - Visual switching between Spring, Summer, Fall, and Winter
- **Save Management** - Support for saving and importing/exporting layout designs
- **Multi-language Support** - Complete Chinese and English interface
- **Online Collaboration** - Share and discuss layout designs through links
